Crée un nouvel abonnement via FrmAbonnementRevuesController.CreerAbonnement en se basant sur la revue sélectionnée dans cbRevue et les données saisies dans les contrôles de souscription.

  • Validation au niveau des dates, du format des données.
  • Vérifie qu'aucun abonnement actif similaire (pour la même revue) est déjà existant dans la même période pour éviter le chevauchement.
  • Demande une confirmation à l'utilisateur pour valider le processus.
